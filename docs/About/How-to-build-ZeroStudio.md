Build a construction environment：

Hardware requirement: At least 8GB or more of RAM should be prepared，Add 16GB of storage.
During the construction period, at least 6-8 GB of running memory should be consumed


Recommended system: Ubuntu 22.04 and above versions of Ubuntu system

Java environment: JDK17, JDK21 and above environments

Install package：Android platform building tools, ndk27，cmake3.31，3.2+

Software package installation：git，python3.11，proto Build tool，gcc，ollvm，gradle....
There are quite a few software packages that need to be installed, which are probably necessary as mentioned above.
